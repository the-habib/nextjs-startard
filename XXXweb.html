1 • Typography

The foundation of great web design

We use a modular scale for headings, generous line-height for readability, and keep line-length near 65–70ch. Headings carry the weight; body stays calm. This yields predictable rhythm across every section.

clamp()70ch measure-0.02em tracking

Type scale in action

H1 → H2 → Body creates clear scanning patterns and hierarchy.

  • • H1: product promise
  • • H2: section purpose
  • • Lead/body: focused, scannable, specific

2 • Layout

Perfecting structure, spacing & visual hierarchy

We standardize vertical rhythm with .section spacing, use clusters for button groups, and grid utilities for responsive blocks. No hard borders; sections blend with soft separators.

Consistent rhythm

Clamp-based section padding and stack spacing keep flow consistent.

Grid clarity

Two- and three-column grids adapt from mobile to desktop cleanly.

Scan patterns

Headline → subhead → action. Cards emphasize key benefits.

4 • Coding

Essential skills for web designers

Semantic landmarks, focus-visible styles, ARIA labels, prefers-reduced-motion support, and meaningful contrast. The markup remains simple, readable, and production-friendly.

Semantic HTMLA11y focusReduced motion

What’s applied here

  • • Proper landmarks (header, nav, main-like sections, footer)
  • • Buttons/links with clear names & intent
  • • Keyboard-friendly focus outlines
  • • No “infinite animations” if user prefers reduced motion
